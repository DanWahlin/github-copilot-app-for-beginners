# Security

Please help keep this course and its learners safe.

## Reporting a vulnerability

Do not open a public issue for a security concern.

Report security problems privately to the repository owner or maintainer contact listed for this project. Include:

- a short description,
- affected file or workflow,
- steps to reproduce if safe,
- potential impact,
- suggested fix if known.

Do not include real tokens, credentials, customer data, or private organization details in the report.

## Course asset safety

Before contributing screenshots, prompts, canvas examples, automation examples, or terminal output, review:

- [Screenshot Capture Guide](appendices/screenshot-capture-guide.md)
- [Privacy-Safe Course Assets](appendices/privacy-safe-course-assets.md)

Never publish:

- GitHub tokens,
- API keys,
- model provider credentials,
- private repository names,
- customer data,
- organization-specific policy details,
- private server URLs.

## Supported versions

This repository is course material. Security fixes should target the current course content in the default branch unless maintainers document a release branch policy.
